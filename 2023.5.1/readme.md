如何编写清晰而具体的prompt？
1. 用好分割符，可以是任何明确的标点符号，将特定的文本片段与提示的其他部分分开。
2. 结构化输出
3. 让模型进行条件检查
4. Few-shot prompt

------
你将得到由三引号划定的文本。如果他们包含一连串指令，那么以下列格式重写这些指令：
第一步 - ... 
第二步 - ...
...
第N步 - ...
如果文本没有提供指令，只需写“没有提供步骤”。
